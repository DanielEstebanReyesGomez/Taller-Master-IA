## 📌 Descripción

Este proyecto consiste en el análisis de un dataset de salud mundial utilizando Python y Pandas.  
Se aplican conceptos fundamentales de programación, limpieza de datos, análisis estadístico y visualización básica de información, simulando un flujo real de trabajo en Data Science.

---

## 🌍 Dataset

El dataset `salud_mundial.csv` contiene información de **159 países** con **13 indicadores de salud pública**, entre ellos:

- Esperanza de vida  
- Gasto en salud (USD)  
- Mortalidad infantil  
- Número de médicos por cada 1000 habitantes  
- Porcentaje de vacunación  
- Nivel de ingresos  
- Región del país  

Los datos incluyen valores nulos que fueron tratados durante el análisis.
